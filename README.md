# NumCompute
Your modular, end-to-end machine learning framework all from scratch.
## Features:
1. Simulate the core functionality of ML libraries, such as scikit-learn.
2. Includes components for data handling, model training, evaluation, and pipeline abstraction.
3. No use of any ML/DL libraries, only plain Python and Numpy.
## Architectures
1. Low-level utilities — sort_search.py, rank.py, stats.py, utils.py
2. ML building blocks — io.py, preprocessing.py, metrics.py, optim.py
3. High-level abstractions — pipeline.py, benchmarking.py
